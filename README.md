# Kader⁴²
Scripts and configuration-profiles for mkarchiso for creating the Linux distribution Kader⁴². 
This distribution is optimized for convertible notebooks such as the Framework 12.

The goal of this distribution is to provide all the features you’d expect from a distribution on a convertible, right out of the box:
* Display rotation
* The appropriate launcher based on whether you’re in notebook or tablet mode
   * Dash menu in tablet mode
   * Standard menu in notebook mode
* Preconfigured swipe gestures similar to those on Android or iOS
* On-screen keyboard in tablet mode

If any of these features don't work on your device, please feel free to contact me. We can then work together to find a solution, which will eventually be incorporated into the distribution. 
It's possible that there are devices where display rotation won't work properly. In that case, we'll check whether a ready-made hwdb file with the matrix already exists, or we'll have to try to create a corresponding file 

This is a personal project. I treated myself to Framework 12, but unfortunately found that it doesn’t work very well with Linux and convertibles yet. That’s how this project came about. As a result, I can only cite Framework 12 as a working reference. If you’re able to use the system successfully on other convertibles without any modifications, please feel free to let me know. I would then compile a corresponding list of “tested” machines.
